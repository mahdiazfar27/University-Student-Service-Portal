## Project Overview

### Objective  
The **University Academic Management System (UAMS)** is a web-based platform designed to manage and automate academic operations of a university.
The system aims to replace traditional manual processes with a **centralized digital solution** for handling student records, courses, results, and academic activities.
By providing role-based access for administrators, teachers, and students, UAMS ensures **accuracy, efficiency, transparency, and data security**.
The project focuses on improving academic workflow management and reducing administrative overhead.

### Target Audience  
- University **students** who need access to courses, grades, results, and academic information  
- **Faculty members** for managing courses, attendance, and academic performance  
- **University administrators** responsible for maintaining student, teacher, and course data

## Project Features

### Core Features
- **Role-based login system** (Admin, Teacher, Student)  
- **Student information management**  
- **Teacher profile and course assignment management**  
- **Course and department management**  
- **Result and transcript management**  

### Advanced Features
- **Automated GPA and CGPA calculation**  
- **Attendance tracking system**  
- **Course enrollment and registration module**  
- **Academic performance reporting**  

---

### Authentication
- **JWT-based secure authentication**  
- **University email-based user verification**  
- **Secure password hashing and access control**

---

### CRUD Operations
- **Student Module:** Create, view, update, and delete student records  
- **Teacher Module:** Create, view, update, and delete teacher profiles  
- **Course Module:** Full CRUD operations for courses and departments  
- **Result Module:** Create, view, update, and delete academic results  
- **User Management:** Admin-controlled role assignment
